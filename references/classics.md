# 引用古籍档案

本档案收录 README 中提到的六部古籍，记录其基本信息、核心概念，以及对活动建模的参考价值。

---

## 《史记》

- **编著：司马迁**
- **成书：西汉**
- **内容简介：**纪传体通史著，以人物为中心，分本纪、表、书、世家、列传五体，记录从黄帝到汉武帝约三千年的重大政治、军事、外交活动的过程与结果。**
- **关键概念：**
  - 五体结构：本纪（时间轴/里程碑）、表（对照矩阵）、书（制度与流程记录）、世家（组织传承）、列传（关键干系人）
  - 究天人之际，通古今之变：强调从长周期视角审视事件因果
  - 决策案例：鸿门宴（风险评估）、卧薪尝胆（目标管理）、商鞅变法（变革管理）、四面楚歌（信息战）
- **建模参考：**可视为古代大型活动的案例库，提供了复杂活动中多因素交织的因果链，以及决策节点如何影响最终成败的过程数据。**

## 《周易》

- **编著：传说伏羲画卦、周文王演卦、孔子作传（十翼），实为集体创作**
- **成书：《易经》约西周至春秋，《易传》约战国**
- **内容简介：**以阴阳、八卦、六十四卦为符号系统，推演宇宙万物变化规律，从卜策之书升华为哲学经典，核心在"变易"与"不易"。**
- **关键概念：**
  - 元亨利贞：事物发展的四阶段（创始、通达、和谐、正固）
  - 吉凶悔吝：四种结果状态，对应风险管理中的成功/失败/补救/隐患
  - 时、位、中、应：活动中的时机、位置、适度、关联四维度
  - 穷则变，变则通，通则久：变革管理与流程优化
  - 象、数、理：现象、数据、规律三层分析框架
- **建模参考：**提供了最古老的情境建模与阶段论框架。"元亨利贞"可直接映射活动生命周期；"时中"强调过程控制中的时机把握；六十四卦可视为六十四种情境模式，用于活动前期的情境推演。**

## 《孙子兵法》

- **编著：孙武**
- **成书：春秋末期**
- **内容简介：**共十三篇，系统论述战争规律、战略谋划、战术执行与情报保障，主张"不战而屈人之兵"的全胜思想。**
- **关键概念：**
  - 道天地将法（五事）：目标共识、天时、地利、将领、制度/流程
  - 七计：主止有道、将止有能等，类似现代 SWOT/竞争力分析
  - 未战先算，多算胜少算不胜：强调规划阶段的可行性计算
  - 全胜与破胜：目标管理中的最优解与次优解
  - 奇正、虚实：过程控制中的资源灵活调配与路径变通
  - 九地：不同情境下的行为模式
  - 用间：信息收集与干系人管理
- **建模参考：**古代最严密的活动管理方法论。从"计篇"的目标论证，到"作战篇"的资源估算，再到"九变篇"的风险预案，构成完整的"规划—执行—控制—收尾"闭环，可直接映射到现代项目管理流程。**

## 《鬼谷子》

- **编著：鬼谷子（王詹）**
- **成书：战国**
- **内容简介：**纵横家代表作，核心研究人际互动、说服游说、情报摸捉与谋略决策，重在"摸捉人心"与"权衡局势"。**
- **关键概念：**
  - 掠阖（开合）：活动节奏控制，信息的开放与封闭策略
  - 反应：信息反馈与验证机制（反以观往，覆以验来）
  - 内揕：与关键干系人建立信任与沟通渠道
  - 抵嶋：风险早期识别与缝隙修补（危机管理）
  - 飞箍：激励与约束并用（干系人管理）
  - 揃、摸、权、谋、决：信息收集→心理分析→权衡→谋划→决策的五步决策流程
- **建模参考：**提供了以"人"为核心的活动交互模型。现代活动管理中沟通、谈判、利益协调占比极高，"揃情摸意"对应需求分析，"掠阖"对应沟通策略，"决篇"提供五种决策准则，对干系人管理和决策节点设计极具启发。**

## 《管子》

- **编著：托名管仲，实为管仲学派及稷下学者汇编**
- **成书：战国至西汉初**
- **内容简介：**内容涵盖政治、经济、军事、哲学，是一部国家治理百科全书，特别强调富国强兵、资源调控与社会分工。**
- **关键概念：**
  - 轻重：通过调控货币与物资的"轻重"关系实现宏观调控，对应现代资源分配与成本控制
  - 四民分业：士农工商的职业化分工与组织管理
  - 七法：则（规律）、象（表象）、法（制度）、化（教化）、决塞（疏导与管控）、心术（领导方法）、计数（数据核算）
  - 仓廎实而知礼节，衣食足而知荣辱：物质基础是活动前提
  - 四时五行：按时间节律安排生产与行政活动
- **建模参考：**古代最大型的"国家治理项目"管理手册。"轻重"理论是早期的系统动力学与资源杠杆思想；"计数"强调用数据核算支持决策；"四民分业"是组织架构设计的雏形。对大型复杂活动的资源配置、成本预算、时序规划有直接参考意义。**

## 《大学》

- **编著：传为曾子（曾参），原为《礼记》一篇**
- **成书：战国**
- **内容简介：**儒家修身治国平天下的纲领性文献，提出从个人修养到国家治理的系统化路径。**
- **关键概念：**
  - 三纲领：明明德、亲民、止于至善——目标体系的三个层次
  - 八条目：格物、致知、诚意、正心、修身、齐家、治国、平天下——从信息获取到最终交付的完整价值链/工作分解结构
  - 知止而后有定：目标明确（知止）是规划与执行的前提
  - 物有本末，事有终始，知所先后，则近道矣：强调流程顺序、关键路径与先后依赖关系
  - 慎独：无人监督时的过程自律与质量控制
  - 索矩之道：推己及人的协调方法（干系人换位思考）
- **建模参考：**提供了最清晰的"目标—手段"分解链（八条目），可作为活动 WBS 的古典原型。"事有终始"是流程管理思想的源头；"知止"对应范围管理；"格物致知"对应数据收集与需求分析。将个人、家庭、国家三个层级的活动统一在一个逻辑框架内。**
